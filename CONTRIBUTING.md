# Contributing to tt-cluster-descriptors

Thank you for your interest in contributing!

## Getting Started

- Fork the repository and create your branch from `main`.
- Make your changes.
- Open a pull request with a clear description of the change.

## Pull Request Guidelines

- One logical change per PR.
- Include a brief description of what changed and why.
- Reference any relevant issues or tickets.

## Code Style

This repository contains static YAML/JSON cluster descriptor files. Please ensure:

- Files are valid YAML/JSON (use a linter before submitting).
- New descriptor files follow the naming conventions of existing files.
- Any new cluster topology is accompanied by a brief comment or README note explaining the hardware configuration.
